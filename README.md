## Features

| | |
|---|---|
| 👥 | Add 1–3 players and up to 4 rounds |
| ⌨️ | Intuitive score entry with keyboard navigation |
| 🧮 | Automatic round totals and relative score calculation |
| 📊 | Results view with per-player averages |
| 📤 | Export scores as CSV or share via system share sheet |
| 🌗 | Full light and dark mode support |

## Getting Started

### Requirements

- iOS 15+
- Xcode 14+

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/willegyr/MinigolfScorecardApp.git
   ```

2. Open `Minigolf.xcodeproj` in Xcode.

3. Build and run on a simulator or physical device.
